    Hill numbers

    The Hill numbers expression was first discovered by Hill in 19731, hence the use of the name “Hill numbers”. This family of diversity metrics has two major advantages over diversity indices such as Shannon and Simpson:

    1. The interpretation of the measure and its measurement unit is always the same. Hill numbers yield diversity measures in “effective number of types”, that is, the number of equally abundant or frequent types that would be needed to give the same value of diversity2.
    2. The sensitivity towards abundant and rare types can be modulated with the parameter q, i.e. the order of diversity3.

    The Hill numbers expression

    \( ^qD = (\sum_{i=1}^S{p_i^q})^{(\frac 1 {1-q})} \)

    in which pi is XXX, S is XXX and q is XXX

    The Hill numbers expression is not defined for q=1, but it is for its limits (e.g. 0.999999). Researchers can therefore opt for using such an approximation, or using this second expression to find the exact value:

    \( ^1D = exp(-\sum_{i=1}^S(p_i * ln(p_i))) \)

    Computing Hill numbers with R

    Although the expression might seem complicated, the following R code shows that it is actually rather simple:

    sum(pi^qvalue)^(1/(1-qvalue))

    Let’s see an example:

    #Declare vector of relative frequencies that sums up to 1
pi <- c(0.12,0.34,0.03,0.41,0.10)
sum(pi)
# 1

#Declare q-value (modify q-value to the limit of the unity if q=1)
qvalue=1
if(qvalue == 1){qvalue = 0.9999999}

#Compute Hill number
sum(pi^qvalue)^(1/(1-qvalue))
# 3.751806

    Order of diversity q

    The sensitivity towards abundant/non-abundant (if abundance-based) or frequent/unfrequent (if incidence-based) types can be modulated using the scaling parameter q, known as the “order” of diversity2. The larger the q value, the higher the importance attributed to abundant or frequent types. Three q values are particularly relevant, both for their significance, and their close relationship to popular diversity indices: q = 0, q = 1 and q = 2.

    q=0

    When a diversity of order zero (q = 0) is applied to the formula, it becomes insensitive to type frequencies, thus yielding a richness value. As the relative abundances of types are overlooked, rare types are overweighed.

    Check how the Hill number of q=0 is identical to the number of types, i.e. type richness.

    qvalue=0
sum(pi^qvalue)^(1/(1-qvalue))
# 5
length(pi)
# 5

    q=1

    A q value of 1 (in practical terms its limit, as the Hill number is undefined for q = 1) is the value that weighs types by their frequency, without disproportionately favouring either rare or abundant ones. The value it yields is exactly the exponential of the Shannon index. In fact, q values under unity favour non-abundant or unfrequent types, while values above one favour abundant or frequent types4.

    Check how the Hill number of q=1 can be obtained by applying a limit of the unity to the Hill number formula, or by computing the exact value. Note that the closer the limit is to the unity the approximation becomes more accurate.

    #Using the limit approximation
qvalue=0.99999999
sum(pi^qvalue)^(1/(1-qvalue))
# 3.751806

#Computing the exact value
exp(-sum(pi*log(pi)))
# 3.751806

#Using the limit approximation with fewer decimals
qvalue=0.9999
sum(pi^qvalue)^(1/(1-qvalue))
# 3.751883

    q=2

    When a q value of 2 is applied, abundant or frequent types are overweighed, and the formula yields the multiplicative inverse of the Simpson index.

    Computing Hill numbers using the R package hilldiv

    Simple, right?

    hill_div(pi,qvalue=0)
# 5
hill_div(pi,qvalue=0.5)
# 4.240513
hill_div(pi,qvalue=1)
# 3.751814
hill_div(pi,qvalue=2)
# 3.236246

    Relationship with the Shannon index

    \( ^1D = exp(^1H) \)

    See how the Hill number of q=1 is the exponential of the Shannon index.

    #Shannon index using formula
-sum(pi*log(pi))
#1.32223

#Shannon index using hilldiv
index_div(pi,index="shannon")
# 1.3223

#Exponential of the Shannon index
exp(index_div(pi,index="shannon"))
# 3.7518

#Hill number of q=1
hill_div(pi,qvalue=1)
# 3.7518

    Relationship with the Simpson index

    \( ^2D = 1/(1-^2H) \)

    See how the Hill number of q=2 is the multiplicative inverse of the Simpson index.

    #Simpson index using formula
1-sum(pi^2)
# 0.691

#Shannon index using hilldiv
index_div(pi,index="simpson")
# 0.691

#Multiplicative inverse of the Simpson index
1/(1-(index_div(pi,index="simpson")))
# 3.236246

#Hill number of q=2
hill_div(pi,qvalue=2)
# 3.236246

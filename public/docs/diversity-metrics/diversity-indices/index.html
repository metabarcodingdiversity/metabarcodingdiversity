    Shannon index

    Text to be added1.

    \( ^1H = -\sum_{i=1}^S(p_i * ln(p_i)) \)

    Calculate Shannon index using R

    #Declare vector of relative frequencies that sums up to 1
pi <- c(0.12,0.34,0.03,0.41,0.10)
sum(pi)
# 1

#Shannon index using formula
-sum(pi*log(pi))
# 1.322237

#Shannon index using hilldiv
index_div(pi,index="shannon")
# 1.322237

#Shannon index using vegan
diversity(pi,index="shannon")
# 1.322237

    Simpson index

    Text to be added.

    \( ^2H = 1-\sum_{i=1}^S(p_i^2) \)

    Calculate Simpson index using R

    #Shannon index using formula
1-sum(pi^2)
# 0.691

#Shannon index using hilldiv
index_div(pi,index="simpson")
# 0.691

#Shannon index using vegan
diversity(pi,index="simpson")
# 0.691
